Enables Relative Metrics on all pages.

== Description ==

This plugin adds the required javascript for Relative Metrics and enables automatic reports about your website users statistics.

For more information about Relative Metrics visit:

[Relative Metrics](https://relativemetrics.com)

== Installation ==

1. Upload the `relative-metrics` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the API Key from Relative Metrics to the settings (Admin > Settings > Relative Metrics)
